# ScanMyReceipt

ScanMyReceipt is an iOS application designed to help users scan and manage their receipts efficiently. The app provides a user-friendly interface for capturing receipt data using the device's camera and organizing it for easy access.

## Features

- **Receipt Scanning**: Use your device's camera to scan receipts and automatically extract relevant information.
- **Receipt Management**: View and manage a list of scanned receipts, including details such as date, amount, and items.
- **Detailed View**: Access detailed information about each receipt, making it easy to track expenses.

## Project Structure

The project is organized into several key components:

- **App**: Contains the main entry point and the primary content view of the application.
- **Models**: Defines the data structures used within the app, such as the `Receipt` model.
- **Views**: Includes the various user interface components, such as the list of receipts and the scanner view.
- **ViewModels**: Manages the logic and data for the views, ensuring a clean separation of concerns.
- **Services**: Provides functionality for scanning and data persistence.
- **Utilities**: Contains utility extensions for common tasks.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `ScanMyReceipt.xcodeproj` file in Xcode.
3. Build and run the project on a physical device to test the scanning functionality.
